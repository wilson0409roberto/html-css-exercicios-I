Ferramenta no Google para ver paleta de cores Num site em desenvolvimento se usa geralmente de 3 a 5 cores numa paleta. Conforme você escolhe a paleta, abaixo da paleta, em cada cor, ele fornece o código hexadecimal de cada cor selecionada. A ferramente permite que você coloque o código hexadecimal de uma cor e a partir dela a ferramenta forma a paleta que selecionarmos para fazer, de forma harmoniosa. Na opção de 'Extrair tema' da ferramenta, você obtém as cores de uma imagem formando uma paleta. Em 'Explorar'/'Exibir', tem a opção de 'Temas de cores' e a opção de 'Mais usados', para verificar as paletas mais usadas. Clicando na paleta temos a opção de copiar como CSS. Outra ferramenta para ver paleta de cores. Após escolher a quantidade de cores, selecionar as cores e na opção de 'Exemplo' você obterá uma sugestão de site com as cores escolhidas.
